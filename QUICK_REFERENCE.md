
## ⏱️ 停留时间阈值

## 📊 核心算法

### 任务生成
```
10天3000词 →
Day 1: 300新词 + 0复习 = 3,000次
Day 2: 300新词 + 20复习 = 3,100次
...
```

### 复习选择
```
SELECT * FROM word_learning_records
WHERE day < current_day
ORDER BY avg_dwell_time DESC  ⭐
LIMIT 20
```

### 队列生成
```
300新词 × 10次 = 3,000
20复习 × 5次 = 100
总计 = 3,100次滑卡
随机打乱（避免连续）
```
